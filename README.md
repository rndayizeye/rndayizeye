## 📌 Featured Projects

### 🤖 [RAG STI Surveillance Assistant](https://github.com/rndayizeye/rag_surveillance_demo)
An AI assistant for STI disease intervention. Uses Retrieval-Augmented Generation (RAG) to let public health staff query CDC guidelines, investigation protocols, and program manuals — covering both guidelines and case investigation workflows — in plain language. Built with Python and Streamlit. **[Try it live on HuggingFace →](https://rndayizeye-rag-surveillance-pro.hf.space)**

### 🩸 [Predicting Intraoperative Blood Transfusion](https://github.com/rndayizeye/blood_tranfusion)
ML models trained on real-world VitalDB surgical data to predict intraoperative transfusion need from preoperative variables — reducing blood product waste. Fully Dockerized with MLOps best practices. Built in R.

### 🚗 [NAVIGATOR: NYC Crash & Health Equity Analysis](https://github.com/rndayizeye/navigator_tdsp) *(in progress)*
Geospatial + time series analysis of 2M+ NYC collision records — poverty-crash correlation by ZIP code via Census API, Prophet forecasting, change point detection, interactive Folium choropleth maps. In collaboration with U.S. DOT Federal Highway Administration.

### 📊 [Pens & Printers: Sales Method Optimization](https://github.com/rndayizeye/pens_and_printers)
End-to-end business data analysis on 15,000 observations — ANOVA with Bonferroni correction, effect size analysis (η²=0.54), KPI design, and executive recommendation. DataCamp Certified Data Analyst capstone project.

### 🗺️ [NYC Gentrification & Health Equity Analysis 2000–2010](https://github.com/rndayizeye/gentrification_nyc_2000_2010)
Geospatial analysis of neighborhood change in NYC as a social determinants of health case study — using the same spatial cluster methods applied in real STI/HIV morbidity hotspot analysis.
